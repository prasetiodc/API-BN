# API-BN

## **User Route**

- **Signup**
  - URL:
    - `POST /user/signup`
  - Body:
    - `nik`: `String`, required
    - `name`: `String`, required
    - `email`: `String`, required
    - `password`: `String`, required (include 6 length, 1 Capital, 1 Small, 1 Symbol)
    - `role_id`: `String`, (default MD) 
  - Expected response :
    - Success (status: `201`)
      ```json
        {
          "message": "Success",
          "data": {
            "id": "...",
            "nik": "...",
            "name": "...",
            "email": "..."
          }
        }
      ```
    - Error (status: `400, 500`)

- **Signin**
  - URL:
    - `POST /user/signin`
  - Body:
    - `email`: `String`, required
    - `password`: `String`, required
  - Expected response :
    - Success (status: `200`)
      ```json
        {
          "message": "Success",
          "data": {
            "token": "...",
            "user_id": "...",
            "name": "...",
            "role_id": "..."
      ```
